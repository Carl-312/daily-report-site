## 归档规则

1. 先把仍有价值的当前结论提炼到 active 文档，再移动原文。
2. 归档文件保留日期、背景和原始链接；开头标注 canonical 当前入口。
3. 不在归档文档中继续维护“当前状态”；若发现结论变化，只更新 active 文档并补充历史说明。
